### Instalasi

1.  **Clone repository ini:**
    ```sh
    git clone [URL-REPOSITORY-ANDA]
    cd [NAMA-FOLDER-REPOSITORY]
    ```

2.  **Buat dan aktifkan virtual environment (dianjurkan):**
